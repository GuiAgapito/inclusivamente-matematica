# InclusivaMente Matemática

Ambiente 3D interativo voltado ao ensino de matemática básica para alunos neurodivergentes, com foco em visualização, manipulação de objetos e aprendizagem por interação.

---

## 📌 Visão Geral

O projeto propõe uma aplicação web utilizando computação gráfica para facilitar o entendimento de conceitos matemáticos fundamentais por meio de elementos visuais e interativos em 3D.

**Nome curto:** InclusivaMente Matemática

---

## 🎯 Objetivo Técnico

- Uso de WebGL com Three.js  
- Importação de modelos 3D (.glb / .gltf)  
- Iluminação com sombras  
- Aplicação de materiais e texturas  
- Interatividade com o usuário
